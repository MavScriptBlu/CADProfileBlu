Project Overview

PixelBoof is a virtual photo booth application where users use a virtual camera to snap a photo, and then generative AI creates a unique original character based on the image. The app is deployed on Google Cloud Run for scalable, serverless hosting.

Key Features
  • Virtual camera for photo capture
  • Generative AI character creation from user photos
  • Unique, one-of-a-kind character output every time
  • Deployed on Google Cloud Run
  • Responsive web interface

WTWR (What to Wear)
Project Overview

WTWR is a full-stack web application that helps users decide what to wear based on current weather conditions. The React frontend fetches weather data and presents clothing suggestions, while the Express backend handles user data, authentication, and clothing item management.

Key Features
  • Real-time weather data integration
  • Clothing suggestions based on temperature
  • User accounts with authentication
  • Add, edit, and manage clothing items
  • Responsive UI built with React
