# MavScript.blu - Personal Portfolio

A responsive personal portfolio website built with **Bootstrap 5** and **jQuery**, showcasing my projects, skills, and experience as a Student Software Engineer and Freelance Web Developer.

## Live Preview

Open `index.html` in any modern browser to view the site.

## Theme

**Personal Portfolio** - A dark-themed, single-page portfolio featuring real projects from my GitHub repositories, animated skill bars, project filtering, and a contact form.

## Tech Stack

- **Bootstrap 5.3.2** - Responsive layout, components, and grid system
- **jQuery 3.7.1** - DOM manipulation, animations, and interactivity
- **Bootstrap Icons 1.11.3** - Iconography throughout the site
- **Google Fonts** - Inter (body) + Playfair Display (headings)
- **Custom CSS** - Dark theme with CSS custom properties
- **Custom JavaScript** - jQuery-powered interactions and effects
